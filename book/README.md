# EcoFin 开源书

这是 Obsidian Vault 的根目录，包含完整的金融经济百科知识库。

## 如何使用

### 在 Obsidian 中打开

1. 打开 Obsidian 应用
2. 点击 "Open folder as vault"
3. 选择此目录（`book/`）
4. 开始探索！

- **主入口**：[[00-index|知识地图]]
- **经济学**：[[01-economics|经济学篇]]
- **金融学**：[[02-finance|金融学篇]]
- **政治学**：[[03-politics|政治学篇]]
- **法学**：[[04-law|法学篇]]
- **会计学**：[[05-accounting|会计学篇]]
- **经济史**：[[06-history|经济史篇]]

## 特色

- 六学科知识体系
- 中美对比视角
- 五段式结构（理论→数学→对比→直觉→实践）
- 篇章骨架已建立，模块级扩充持续推进

## 当前状态

- `00-index`、`01-economics`、`APP-investment` 已发布为稳定版
- 其余篇章仍为 `draft`，会继续补足模块深度与校对
- 模块扩充进度见 [../docs/tracking/模块清单.md](../docs/tracking/模块清单.md)
