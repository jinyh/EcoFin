# 金融经济百科知识库

> 从基础学科到投资实践：构建中美视角的完整认知框架

## 项目定位

本知识库采用百科综合视角，既包含经济学、金融学的理论框架，也提供中美对比视角下的实践指南。适合：
- 投资者：理解市场机制、掌握资产配置方法
- 政策研究者：分析经济政策与制度框架
- 金融从业者：构建完整知识体系、规划职业发展

## 知识地图

| 编号 | 学科 | 核心问题 |
|------|------|----------|
| [01](./01-economics.md) | 经济学 | 资源配置、市场机制、宏观调控 |
| [02](./02-finance.md) | 金融学 | 价值配置、风险管理、金融工具 |
| [03](./03-politics.md) | 政治学 | 制度框架、政策制定、权力博弈 |
| [04](./04-law.md) | 法学 | 规则保障、合约执行、产权界定 |
| [05](./05-cross-themes.md) | 交叉主题 | 金融与法律、经济与政治的跨学科议题 |

**实践入口**：[投资实践附录](./APP-investment.md)

## 特色

- **中美对比视角**：每个知识点同时呈现中美差异
- **五段式结构**：理论 → 数学表述 → 中美对比 → 直觉理解 → 实践指南
- **Obsidian 兼容**：支持双向链接、知识图谱可视化

## 使用建议

1. **首次阅读**：从 [00-index.md](./00-index.md) 了解知识库结构
2. **投资者路径**：金融学 → 经济学 → 法学 → 投资附录
3. **政策研究者路径**：经济学 → 政治学 → 法学
4. **金融从业者路径**：金融学 → 法学 → 经济学
