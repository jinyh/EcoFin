# 金融经济百科知识库

> 从基础学科到投资实践：构建中美视角的完整认知框架

## 项目定位

本知识库采用百科综合视角，包含六学科体系，提供中美对比视角下的理论与实践指南。适合：
- 投资者：理解市场机制、掌握资产配置方法
- 政策研究者：分析经济政策与制度框架
- 金融从业者：构建完整知识体系、规划职业发展

## 特色

- **六学科闭环**：经济→金融→政治→法律→会计→历史，形成完整知识体系
- **中美对比视角**：每个知识点同时呈现中美差异
- **五段式结构**：理论 → 数学表述 → 中美对比 → 直觉理解 → 实践指南
- **Obsidian 兼容**：支持双向链接、知识图谱可视化

## 使用建议

1. **首次阅读**：从 [00-index.md](./00-index.md) 了解知识库结构
2. **投资者路径**：金融学 → 会计学 → 经济学 → 法学 → 投资附录
3. **政策研究者路径**：经济学 → 政治学 → 法学 → 经济史
4. **金融从业者路径**：金融学 → 会计学 → 法学 → 经济学

## 版本管理

采用三段式版本号：`v[major].[minor].[patch]`

- Minor：知识点新增或内容扩充
- Patch：格式修正或 typo 修复
