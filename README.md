# EcoFin：金融经济百科（中美视角）

> 六学科知识体系，支持Obsidian双向链接

## 项目定位

本知识库采用百科综合视角，包含六学科体系，提供中美对比视角下的理论与实践指南。适合：
- 投资者：理解市场机制、掌握资产配置方法
- 政策研究者：分析经济政策与制度框架
- 金融从业者：构建完整知识体系、规划职业发展

## 使用方式

### Obsidian 用户

1. 将 `book/` 目录作为 Obsidian Vault 打开
2. 从 `00-index.md` 开始浏览
3. 使用双向链接 `[[文件名|显示文本]]` 导航
4. Graph View 可视化知识图谱

### GitHub 阅读

直接浏览 `book/` 目录下的 Markdown 文件。

## 特色

- **六学科闭环**：经济→金融→政治→法律→会计→历史，形成完整知识体系
- **中美对比视角**：每个知识点同时呈现中美差异
- **五段式结构**：理论 → 数学表述 → 中美对比 → 直觉理解 → 实践指南
- **Obsidian 兼容**：支持双向链接、知识图谱可视化

## Obsidian 使用指南

### 打开 Vault

1. 打开 Obsidian
2. 选择 "Open folder as vault"
3. 选择 `book/` 目录
4. 开始浏览

### 双向链接

- `[[01-economics]]` → 链接到经济学篇
- `[[01-economics|经济学]]` → 显示为"经济学"
- Graph View → 可视化所有链接关系

### 推荐插件

- **Dataview**: 动态查询和展示
- **Graph Analysis**: 知识图谱分析
- **Tag Wrangler**: 标签管理

## 版本管理

采用三段式版本号：`v[major].[minor].[patch]`
- Major：架构重组或学科增删
- Minor：知识点新增或内容扩充
- Patch：格式修正或 typo 修复
