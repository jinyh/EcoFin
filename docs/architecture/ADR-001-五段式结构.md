# ADR-001: 采用五段式知识点结构

## 状态
已接受

## 背景
金融经济知识库需要统一的知识点呈现格式，既要保证理论严谨性，又要兼顾实践可操作性。传统教材往往理论与实践脱节，中美对比视角也缺乏系统化呈现。

## 决策
采用五段式结构呈现每个核心知识点：
1. 理论定义：学术定义、核心特征
2. 数学化表述：公式、变量说明
3. 中美对比表格：制度差异、数据对比
4. 直觉理解：通俗解释、类比说明
5. 实践指南：投资建议、分析要点

## 后果
**优势**：
- 结构统一，便于学习和查阅
- 理论与实践结合，满足不同读者需求
- 中美对比系统化，突出项目特色

**劣势**：
- 编写工作量较大
- 部分知识点可能不适合完整五段式（可灵活调整）

## 替代方案
1. 传统教材式：纯理论叙述（缺乏实践指导）
2. 纯实践式：只讲操作（缺乏理论基础）
3. 问答式：FAQ 格式（不够系统）
